## 5. Data Storage and Security

### 5.1 Where Data Is Stored
- **Primary database:** MongoDB Atlas cloud infrastructure
- **Local device storage:** Authentication token (JWT) stored in AsyncStorage
- **AI service:** Self-hosted service processing data in real-time

### 5.2 Security Measures
- Passwords are **hashed using bcrypt** (one-way hashing) before storage — we never store plain-text passwords
- Authentication uses **JSON Web Tokens (JWT)** with configurable expiration
- API communication uses **HTTPS** encryption in transit
- MongoDB Atlas provides **encryption at rest** for stored data
- Payment processing is handled entirely by **Razorpay** (Android) and **Apple** (iOS) — we do not store credit card or bank details
- Rate limiting is applied to OTP and authentication endpoints to prevent brute-force attacks
- Razorpay webhooks are verified using **HMAC signature validation**
- Apple receipts are verified server-side with **Apple's verification servers**

### 5.3 Data Breach Notification
In the event of a data breach that affects your personal information, we will notify affected users within 72 hours of becoming aware of the breach, in accordance with applicable law (including GDPR Article 33).
